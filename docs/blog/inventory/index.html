Primary PC

Description

My primary computer is an android phone running DexMode or running a Taskbar overlay with Free Form Floating Windows enabled. It connects to a portable monitor as a display. I learned these monitors are called "lapdocks".

My VPS is accessed through a Tailscale VPN instance. I use SSH and RDP to remotely access the box.

I game over the cloud, split screen, using the GeoForce now application. A second powered USB C to HDMI is used to connect to a larger TV Screen. Up to 6 devices can connect to bluetooth.
